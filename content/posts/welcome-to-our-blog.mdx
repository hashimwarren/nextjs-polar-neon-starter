This is the first post in our new blog built with Next.js 15, Tailwind CSS, and MDX.

## Features

- **Next.js 15** with App Router
- **Tailwind CSS v4** for styling
- **shadcn/ui** components
- **MDX** for content
- **Better Auth** for authentication
- **Polar.sh** for subscriptions
- **Resend** for newsletters

## Getting Started

This starter includes everything you need to build a modern blog with paid newsletter functionality.

### Content Management

Blog posts are written in MDX and stored in the `content/posts` directory. Each post supports:

- Frontmatter metadata
- Rich content with MDX
- Tagging system
- Publication status

### Newsletter

Newsletter issues are stored in `content/newsletter` and can be marked as paid content for subscribers only.
